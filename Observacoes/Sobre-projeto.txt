- Cadastro de usuários

- Cadastro de Tags (elogios possiveis)
    - Somente usuario administrador

- Cadastro de elogios

    - ID do usuario - que está enviando o elogio 
    - ID da Tag
    - Data de criação

- Autenticação de usuario

- Listagem de usuarios
- Listagem de Tags
- Listagem de elogios por usuario
